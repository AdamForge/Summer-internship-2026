# Day 01

Today I started my iOS project. The project will be a mobile app for university students to help them follow their lessons, attendance, exams, tasks, and study time, and later I might add some new features.

First, I created the basic SwiftUI project structure in Xcode. I separated the code into folders like Models, Views, Components, and ViewModels so the project will be easier to understand and improve later.

I also created the first version of the app interface. The app has five main tabs: Home, Lessons, Exams, Focus, and Tasks. On the Home screen, I added a simple dashboard with mock data for courses, attendance risk, final exam information, and academic tasks.

Today I learned how to organize a SwiftUI project better and how reusable components can make the design cleaner. Next, I will improve the Home screen and make the risk cards more useful and easier to read.
