# Day 11

Today I added a new exam detail screen to my iOS app.
Before this, the Exams page only showed exam cards in a list. Now I can tap on an exam card and open a separate detail page for that exam. This page shows more information like the final countdown, vize score, passing grade, required final score, and the vize/final weights.

I also added a small explanation about how the required final score is calculated. I think this is useful because the student can understand where the number comes from, not only see the result.

Today I practiced using NavigationLink in SwiftUI and learned how to create a separate detail screen for one item. Next, I will keep improving the exam section and make it more useful.
