# Day 03

Today I worked on the Lessons page of my iOS app. Before, this page was just a simple list of courses, so I wanted to make it more useful.

I added a small summary at the top of the page. It shows how many courses there are, how many courses have attendance tracking, and how many courses are risky.

Then I improved the course cards. Now each card shows the course name, attendance status, risk badge, progress bar, and how many absences are left. I also made repeated courses different, because attendance is not counted for them.

Today I practiced building better list screens in SwiftUI. Next, I will create a course detail screen, so the user can tap a course and see more information.
